Jesse Ziegler Portfolio for AENG 110
                   
















For this project we had to create a bookmark for elementary school. The first step of this process was to create five thumbnail sketches of a few ideas we had. After that we had to pick one design and create a rough layout with type specifications. We then took this rough layout and created our final design on indesign. We then printed multiple sheets of bookmarks on the printing press. From there all we had to do was cut each bookmark down to size and we had all of our bookmarks. Then we cleaned up the press and sent out the bookmarks!
